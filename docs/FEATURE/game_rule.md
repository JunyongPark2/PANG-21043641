# 게임 룰 상세

## 캐릭터

- 화면 하단에 위치하며, 좌우로만 이동 가능하다.
- 위쪽으로 작살(무기)을 발사할 수 있다.
- 무기는 발사 후 화면 상단(또는 장애물)에 닿을 때까지 직선으로 이동한다.

## 풍선

- **이동**: 중력의 영향을 받아 포물선을 그리며 움직이고, 화면 좌우 벽에 닿으면 반사되어 방향을 바꾼다. 바닥이나 장애물에 닿으면 다시 튀어 오른다.
- **크기 단계**: 풍선은 크기에 따라 여러 단계(대/중/소 등)로 나뉜다.
- **분열**: 무기에 맞은 풍선은 한 단계 작은 풍선 2개로 분열된다. 이 분열은 최대 3단계까지 반복될 수 있다.
- **소멸**: 가장 작은 단계의 풍선이 무기에 맞으면 분열 없이 소멸한다.

## 충돌 및 실패 조건

- 캐릭터가 풍선(어느 크기든)에 직접 닿으면 실패 처리된다.
- 제한 시간 내에 스테이지의 모든 풍선을 제거하지 못하면 실패 처리된다.

## 클리어 조건

- 스테이지에 존재하는 모든 풍선(분열된 풍선 포함)을 제거하면 해당 스테이지를 클리어한다.

## 기타 시스템

- 목숨(생명력), 점수, 아이템 등 부가 시스템은 아직 상세화하지 않았으며, 필요 시 별도 섹션으로 확장한다.
