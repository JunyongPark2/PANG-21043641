# 메인 화면

## 개요

게임 실행 시 가장 먼저 보여지는 화면으로, 플레이어가 게임을 시작할 수 있는 진입점 역할을 한다.

## 구성 요소

- **타이틀**: 게임 이름(PANG)을 표시
- **시작 버튼**: 클릭/입력 시 Mission 1로 진입
- **배경**: 팡 특유의 분위기를 살린 정적 또는 애니메이션 배경 (초기 구현에서는 단순 배경으로 대체 가능)

## 동작 흐름

1. 앱 진입 시 메인 화면이 렌더링된다.
2. 플레이어가 시작 버튼을 클릭하거나 정해진 키(예: Enter/Space)를 입력하면 Mission 1 화면으로 전환된다.

## 범위

- 초기 구현에서는 미션 선택, 옵션, 랭킹 등의 부가 화면은 포함하지 않고 "시작하기" 기능만 우선 구현한다.
- 이후 미션 선택 화면, 설정 화면 등이 필요해지면 별도 문서로 확장한다.
