# Mission 1

## 개요

게임의 첫 번째 스테이지로, 팡의 기본 게임 룰([`game_rule.md`](./game_rule.md) 참고)을 그대로 적용하는 튜토리얼 성격의 스테이지이다.

## 난이도 방향성

Mission 1은 플레이어가 기본 조작과 풍선 분열 메커니즘을 익히는 단계이므로, 다른 스테이지 대비 쉬운 난이도로 설계한다.

- 등장 풍선 수: 큰 풍선 1~2개 정도의 소수로 시작한다.
- 풍선 이동 속도: 느린 편으로 설정하여 초반 진입 장벽을 낮춘다.
- 장애물: 없음 (좌우 벽과 바닥만 존재하는 단순한 사각형 화면).

위 수치는 방향성이며, 정확한 값(풍선 개수/속도/제한 시간 등)은 구현 단계에서 밸런스를 보며 확정한다.

## 규칙

기본 게임 룰([`game_rule.md`](./game_rule.md) 참고)을 그대로 적용하며, Mission 1에서 조정하는 값은 제한 시간뿐이다.

- 제한 시간: 다른 스테이지 대비 여유 있게 설정해 실패 위험을 낮춘다.

## 범위

- 클리어 후 다음 미션으로의 연결(미션 2 이상)은 이후 별도 문서에서 다룬다.
