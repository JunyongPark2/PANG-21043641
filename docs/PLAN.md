# PLAN - Phase별 목표

각 Phase는 항상 실행 가능한(`npm run dev`로 직접 플레이해 볼 수 있는) 상태를 목표로 하며, Phase가 끝날 때마다 고객(사용자)이 직접 테스트하고 피드백을 준다. 다음 Phase는 이전 Phase의 피드백을 반영한 뒤 시작한다.

각 Phase의 "테스트 방법"은 그대로 따라 하면 되는 조작 순서이고, "통과 기준"은 그 결과가 맞다/틀리다를 가르는 명확한 조건이다.

관련 문서: [`PRD.md`](./PRD.md), [`FEATURE/main.md`](./FEATURE/main.md), [`FEATURE/game_rule.md`](./FEATURE/game_rule.md), [`FEATURE/mission1.md`](./FEATURE/mission1.md)

---

## Phase 1. 메인 화면

- **목표**: 타이틀과 시작 버튼이 있는 메인 화면을 만들고, 버튼을 누르면 Mission 1 화면(내용은 비어있어도 됨)으로 전환되도록 한다.
- **테스트 방법**:
  1. `npm run dev`로 접속해 첫 화면을 확인한다.
  2. 시작 버튼을 클릭한다.
- **통과 기준**:
  - 첫 화면에 게임 타이틀 텍스트와 시작 버튼이 보인다.
  - 시작 버튼 클릭 시 다른 화면(Mission 1 화면)으로 바뀐다. 클릭해도 화면이 그대로면 실패.

## Phase 2. 캐릭터 이동

- **목표**: Mission 1 화면에 캐릭터를 배치하고, 키보드 좌/우 입력으로 화면 하단에서 좌우로 이동시킨다. 화면 밖으로 나가지 않도록 벽 처리를 한다.
- **테스트 방법**:
  1. 방향키(← →)를 각각 눌러본다.
  2. 왼쪽 화살표를 화면 왼쪽 끝까지 도달할 때까지 누르고 있는다. 오른쪽도 동일하게 반복한다.
  3. 키를 뗀다.
- **통과 기준**:
  - ← 를 누르면 캐릭터가 왼쪽으로, → 를 누르면 오른쪽으로 이동한다.
  - 화면 좌우 끝에 도달하면 캐릭터가 그 자리에서 멈추고, 화면 밖으로 나가거나 사라지지 않는다.
  - 키를 떼면 캐릭터가 즉시 멈춘다(관성으로 미끄러지지 않는다).

## Phase 3. 무기 발사

- **목표**: 정해진 키(예: Space)로 캐릭터가 위쪽으로 작살(무기)을 발사하도록 한다. 무기는 화면 상단에 닿으면 사라진다.
- **테스트 방법**:
  1. Space 키를 한 번 누른다.
  2. 무기가 화면 위로 사라질 때까지 지켜본다.
  3. Space 키를 연속으로 여러 번 눌러본다.
- **통과 기준**:
  - Space를 누르면 캐릭터 위치에서 무기가 위쪽으로 일직선으로 발사된다.
  - 무기가 화면 상단에 도달하면 화면에서 사라지고(에러 없이), 이후에도 계속 발사가 가능하다.
  - 연속으로 눌러도 무기가 누락되거나 게임이 멈추지 않는다.

## Phase 4. 풍선 등장 및 튕기는 물리

- **목표**: 화면에 큰 풍선을 [`mission1.md`](./FEATURE/mission1.md) 기준 수량(1~2개)만큼 등장시키고, 중력의 영향을 받아 포물선으로 움직이며 좌우 벽/바닥에 부딪히면 튕기도록 한다. (아직 터지지는 않음)
- **테스트 방법**:
  1. Mission 1 화면 진입 후 아무 조작 없이 풍선의 움직임을 30초 이상 관찰한다.
  2. 풍선이 화면 좌/우 벽, 바닥에 부딪히는 순간을 각각 확인한다.
- **통과 기준**:
  - 풍선이 화면 밖으로 나가거나, 벽/바닥을 뚫고 지나가지 않는다.
  - 벽에 부딪히면 반대 방향으로, 바닥에 부딪히면 위쪽으로 방향이 바뀐다.
  - 30초 이상 관찰해도 멈추거나 한 지점에 끼어서 움직임이 없어지지 않는다.

## Phase 5. 풍선 충돌 및 분열

- **목표**: 무기와 풍선의 충돌을 감지해 풍선을 제거하고, [`game_rule.md`](./FEATURE/game_rule.md)의 규칙대로 최대 3단계까지 작은 풍선 2개로 분열시킨 뒤 가장 작은 풍선은 소멸시킨다.
- **테스트 방법**:
  1. 큰 풍선을 향해 무기를 발사해 명중시킨다.
  2. 분열된 중간 크기 풍선 하나를 다시 맞춘다.
  3. 그 다음 분열된 가장 작은 풍선을 다시 맞춘다.
  4. 풍선이 없는 허공에 무기를 발사해본다.
- **통과 기준**:
  - 1단계: 큰 풍선을 맞히면 큰 풍선은 사라지고, 그 자리에서 한 단계 작은 풍선 2개가 생긴다.
  - 2단계: 중간 풍선을 맞히면 다시 더 작은 풍선 2개로 나뉜다.
  - 3단계: 가장 작은 풍선을 맞히면 분열 없이 완전히 사라진다.
  - 허공에 쏜 무기는 아무 풍선에도 영향을 주지 않는다(오탐지 없음).

## Phase 6. 승패 판정

- **목표**: 캐릭터와 풍선이 충돌하면 실패, 제한 시간을 초과해도 실패, 모든 풍선을 제거하면 클리어로 처리하고 결과를 화면에 표시한다.
- **테스트 방법**:
  1. (실패-충돌) 일부러 캐릭터를 풍선에 부딪혀본다.
  2. (실패-시간초과) 풍선을 하나도 터뜨리지 않고 제한 시간이 끝날 때까지 기다린다.
  3. (성공) 화면에 있는 풍선을 전부(분열된 풍선까지) 제거한다.
- **통과 기준**:
  - 1번: 충돌 즉시 "실패" 화면/문구가 표시되고 게임이 멈춘다.
  - 2번: 제한 시간이 0이 되는 순간 "실패" 화면/문구가 표시된다.
  - 3번: 마지막 풍선을 제거하는 즉시 "클리어" 화면/문구가 표시된다.
  - 세 경우 모두 결과 화면에 성공/실패 여부가 텍스트로 명확히 구분되어 보인다.

## Phase 7. Mission 1 통합 및 밸런싱

- **목표**: 메인 화면 → Mission 1 플레이 → 클리어/실패 → (재시작 또는 메인으로 복귀)까지 전체 흐름을 연결하고, [`mission1.md`](./FEATURE/mission1.md)의 난이도 방향성(풍선 수, 속도, 제한 시간 등)에 맞춰 수치를 조정한다.
- **테스트 방법**:
  1. 메인 화면부터 시작해 클리어 또는 실패까지 한 번 끝까지 플레이한다.
  2. 클리어/실패 화면에서 재시작(또는 메인으로 복귀) 버튼을 눌러 처음부터 다시 플레이한다.
  3. 총 2~3회 반복 플레이한다.
- **통과 기준**:
  - 메인 화면부터 결과 화면까지 화면 전환 중 끊기거나 멈추는 구간이 없다.
  - 재시작 시 풍선/캐릭터/시간이 초기 상태로 정확히 리셋된다.
  - 별도 조작 실수 없이 평균적으로 클리어까지 도달 가능한 난이도인지 확인한다(체감 난이도는 고객이 직접 판단해 피드백).
